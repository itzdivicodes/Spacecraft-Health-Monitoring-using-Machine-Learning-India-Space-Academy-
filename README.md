# Spacecraft-Health-Monitoring-using-Machine-Learning-India-Space-Academy

This project detects anomalies in spacecraft telemetry data and predicts system risk.

## Features
- Anomaly Detection (Isolation Forest, Autoencoder)
- Risk Scoring System
- Decision Engine
- Interactive Dashboard

## Dataset
NASA CMAPSS Dataset

## Output
- Anomaly detection
- Risk levels
- Corrective actions
